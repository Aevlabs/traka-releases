# Traka

A lightweight desktop time tracker built for focus. Start a timer, tag your work, and see where your time actually goes.

> This repository hosts releases only. Source code lives at [Aevlabs/traka](https://github.com/Aevlabs/traka).

## Download

Go to the [Releases](https://github.com/Aevlabs/traka-releases/releases/latest) page and download the file for your platform.

### Linux

| Format | Who it's for |
|---|---|
| `.AppImage` | Any distro - just download, `chmod +x`, and run |
| `.deb` | Ubuntu, Debian, and derivatives |
| `.rpm` | Fedora, openSUSE, and RHEL-based distros |
| `.pkg.tar.zst` | Arch Linux and derivatives (Manjaro, EndeavourOS, etc.) |

<!-- | Snap Store | `sudo snap install traka` | -->

### macOS

Download the `.dmg` (universal binary - works on both Intel and Apple Silicon). Open the `.dmg`, drag Traka to Applications, and launch it.

> On first launch macOS may warn that the app is from an unidentified developer. Go to **System Settings > Privacy & Security** and click **Open Anyway**.

### Windows

Download the `.msi` installer and run it.

> Windows SmartScreen may show a warning on first run. Click **More info > Run anyway** to proceed.

## Auto-updates

Once installed, Traka checks for updates automatically. You'll get a prompt when a new version is available - no manual re-downloading needed.

## Features

- **Timer** - start/stop tracking with a description, project, and tags
- **Entry log** - full history grouped by day, with inline editing
- **This Week** - daily bar chart, project breakdown, and weekly goal banner
- **Reports** - heatmap, total tracked, focus time, average daily, and longest streak
- **Projects** - create and manage projects with custom colors
